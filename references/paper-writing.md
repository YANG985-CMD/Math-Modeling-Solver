# Competition Paper Writing

## Standard Structure

1. Abstract
2. Problem restatement
3. Assumptions
4. Symbol definitions
5. Model building
6. Solving process
7. Validation and sensitivity analysis
8. Advantages and limitations
9. Conclusion

## Writing Rules

- Write the abstract last
- Keep notation consistent
- Explain why the model fits the problem, not only how it works
- Show at least one robustness or sensitivity check when possible
- Prefer clear tables and labeled figures over dense prose

## Common Risks

- missing unit explanations
- unexplained weighting choices
- equations with undefined symbols
- results reported without interpretation
- conclusions that overclaim beyond the data
